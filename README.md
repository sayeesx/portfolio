# 🌐 Personal Portfolio – sayees.vercel.app

Welcome to the source code of my personal portfolio website, built using **Next.js** and deployed via **Vercel**.

🔗 **Live site:** [www.sayees.vercel.app](https://www.sayees.vercel.app)

This portfolio is a reflection of my journey in tech showcasing who I am, the work I've done, the skills I’ve learned, and the tools I love using. It is responsive, fast, and built with a modern stack to represent my style and growth.

---

## ✨ Key Features

- ✅ **Next.js** for optimized React rendering and routing  
- ✅ **Tailwind CSS** for utility-first responsive design  
- ✅ **Framer Motion** for smooth UI animations  
- ✅ **EmailJS** integration for contact form (no backend needed)  
- ✅ **Live Chatbot** integration powered by my own backend  
  - [Chatbot Repo → sayeesx/chatbot](https://github.com/sayeesx/chatbot)  
  - Hosted on [Render](https://render.com), seamlessly embedded in the website
